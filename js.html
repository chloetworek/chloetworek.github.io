Chloe's Javascript Programs

Here you will find all of the Javascript apps I have written during my time in the CSC106 course at URI. To create these I utilized tools from Kahn Academy.

Tworek Bitmoji

Dancing Bitmoji
This is a bitmoji created using drawing functions based on a headshot of me. The original bitmoji has been animated allowing it to "dance" every time the restart button is clicked.

Bitmoji Race

Dancing Bitmoji - Functions
My original bitmoji, as well as a bunny, are placed on the screen and animated to race across in the x-direction at randomized speeds. This does the same thing as the original dancing program, but was made using functions.

Racing Bitmoji - Functions

Resize Bitmoji
This racing bitmoji program has the same purpose as the original, however, it was coded using functions. This program draws four differently sized and placed bitmojis using functions. It then allows a bitmoji to be placed wherever the mouse is clicked.

Bitmoji Follow

Medical Decision Tree
This program tells the bitmoji to follow the users mouseY and mouseX coordinates, unless it goes off of the bottom of the screen. At that point, it will no longer respond to the mouse and will show the words "Out of bounds." This is a programmed decision tree that responds to yes or no questions. It ultimately leads to a "diagnosis" of sleep apnea based on if the user clicks on yes or no for three questions.

Bitmoji Rain

Roster Program
This program allows the user to move the umbrella left to right on the screen, choosing to block the continuous rain from hitting the bitmoji or not. This program provides a list of the 2022-23 Disky Business Exec-Board and allows users to find some information about each member when clicking on their name.

Smash Program

Tree Jump
This game gives the user 2 minutes to score as many points as possible by clicking on the moving bitmoji. This game requires the user to move the bitmoji with the spacebar and avoid hitting the trees. When reaching the top, 3 points are added to the score, and 2 points at the bottom. Each tree hit subtracts one point.
